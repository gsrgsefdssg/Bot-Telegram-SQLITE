# Bot-Telegram-SQLITE

## RU

# My First Telegram Bot 🤖

Это мой **самый первый Telegram бот**. Написан на `pyTelegramBotAPI` (telebot) с использованием SQLite.

## 📅 История
Бот создан, когда я только начинал изучать Python. С него всё началось!

## 🎯 Возможности
- Регистрация пользователей
- Сохранение имени и пароля в базу данных
- Пошаговый диалог

## 💾 Технологии
- Python 3.x
- pyTelegramBotAPI
- SQLite3

## 🚀 Запуск
pip install pyTelegramBotAPI

python bot.py
